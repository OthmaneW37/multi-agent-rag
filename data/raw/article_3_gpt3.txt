Titre : GPT-3 : Language Models are Few-Shot Learners
Auteurs : Brown et al., 2020
Organisation : OpenAI

Résumé
GPT-3, avec ses 175 milliards de paramètres, représente une avancée qualitative dans les capacités des modèles de langage. Contrairement à BERT qui nécessite un fine-tuning pour chaque tâche, GPT-3 démontre des capacités de few-shot learning, permettant d'accomplir des tâches diverses avec seulement quelques exemples dans le prompt.

Architecture et entraînement
GPT-3 utilise une architecture décodeur-only ( Transformer décodeur ) avec 96 couches, 96 têtes d'attention, et une dimension d'embedding de 12288. Le modèle a été entraîné sur environ 500 milliards de tokens provenant de Common Crawl, WebText2, Books1, Books2 et Wikipedia.

Capacités de few-shot learning
L'article démontre que GPT-3 peut accomplir des tâches sans gradient update :
- Zero-shot : aucun exemple fourni, juste une description de la tâche
- One-shot : un seul exemple dans le contexte
- Few-shot : 10-100 exemples dans le contexte

Sur de nombreuses tâches (traduction, question-réponse, complétion de code, raisonnement arithmétique), GPT-3 few-shot rivalise avec des modèles spécialisés fine-tunés.

Résultats notables
- CoQA F1 : 85.0 (few-shot)
- TriviaQA : 71.2% (zero-shot)
- LAMBADA accuracy : 86.4% (few-shot)
- Codex : capacités de génération de code émergentes

Émergence de capacités
Un phénomène remarquable observé avec GPT-3 est l'émergence de capacités absentes dans les modèles plus petits : raisonnement arithmétique simple, traduction de langues rares, génération de code fonctionnel.

Risques et préoccupations
L'article soulève plusieurs problèmes éthiques :
- Biais sociétaux présents dans les données d'entraînement
- Potentiel de génération de désinformation à grande échelle
- Impact environnemental de l'entraînement (coût estimé à 4.6M$)
- Risques de mémorisation de données privées

Limites
- Hallucinations : génération d'informations factuellement incorrectes
- Coût d'inférence élevé limitant l'accessibilité
- Incapacité à mettre à jour ses connaissances post-entraînement
- Fragilité face aux formulations adversariales des prompts

Conclusion
GPT-3 démontre que l'échelle massive combinée au few-shot learning ouvre de nouvelles perspectives pour l'IA généraliste, tout en soulevant des questions fondamentales sur les biais, la sécurité et l'impact sociétal de ces systèmes.
